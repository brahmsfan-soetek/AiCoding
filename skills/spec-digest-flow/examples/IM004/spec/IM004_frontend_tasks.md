### 表頭欄位 — 異動類型 vs 條件欄位對照

| 異動類型 | 共用欄位 | 額外欄位 |
|---------|---------|---------|
| D01 | 異動類型、過帳日期、文件日期 | 調撥單號(SInput2 → H.REF_DOC_NO) |
| D02 | 異動類型、過帳日期、文件日期 | 成本中心(LOV → H.COST_CODE)、領用申請(SInput2 → H.REF_DOC_NO) |
| D03 | 異動類型、過帳日期、文件日期 | 成本中心(LOV → H.COST_CODE)、報廢原因(SInput2 → H.DESCRIPTION) |
| D04 | 異動類型、過帳日期、文件日期 | 利潤中心(LOV → H.PROFIT_CODE)、收貨原因(SInput2 → H.DESCRIPTION) |
| D05 | 異動類型、過帳日期、文件日期 | 利潤中心(LOV → H.PROFIT_CODE)、發貨原因(SInput2 → H.DESCRIPTION) |

## D. 前端連動邏輯

| # | 項目 | 觸發動作 | 連動結果 |
|---|------|---------|---------|
| F13 | 切換異動類型 | 變更異動類型 Selection | 清除畫面所有資料；切換表頭/表身條件欄位顯示 |
| F14 | 物料 LOV 連動 | 選擇物料 LOV (2.1) | 帶入：物料名稱、單位、單價(AVG_PRICE)、評價類別；即時檢查同單評價類別一致性 |
| F15 | 工廠 LOV 連動 | 選擇工廠 LOV (2.2) | 更新庫位 LOV (2.3) 篩選條件；更新批號 LOV (2.5) 篩選條件 |
| F16 | 庫位 LOV 連動 | 選擇庫位 LOV (2.3) | 帶入庫存水位 (mms.AVL_QTY)；更新批號 LOV (2.5) 篩選條件 |
| F17 | 目標工廠 LOV 連動 | 選擇目標工廠 LOV (2.2) | 更新目標庫位 LOV (2.4) 篩選條件 |
| F18 | 批號 LOV 連動 | 選擇批號 LOV (2.5) | 帶入批號庫存水位 (mmb.AVL_QTY) |
| F19 | 數量輸入連動 | 輸入/變更數量 | 自動計算 金額 = 單價 × 數量 |
| F20 | D01 來源/目標排除 | 來源工廠+庫位已選 | 目標工廠+庫位選項中排除已選來源組合 |

## E. 前端驗證

| # | 項目 | 說明 |
|---|------|------|
| F21 | 必填檢查 | 表頭：異動類型、過帳日期、文件日期；表身：至少一筆明細行、各行必填欄位（物料、數量、工廠、庫位；D01 加目標工廠/庫位） |
| F22 | 庫存水位即時警告 | 出庫數量 > 畫面顯示庫存水位時，前端即時警告（不送後端） |
| F23 | 評價類別一致性 | 選擇物料時即時檢查同單據所有物料 VAL_CLASS 一致，不一致時阻擋 |
| F24 | 批次號條件必填 | ENABLE_BATCH_CONTROL='Y' 時批次號必填 |
| F25 | 已過帳唯讀 | DOC_STATUS='P' 時全部欄位不可編輯，僅可檢視 |
