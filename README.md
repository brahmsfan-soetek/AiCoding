## 核心觀點

- **AI 做執行，人做判斷** — 所有需要判斷的點，都必須在 AI 介入之前由人類完成
- **源頭鐵則** — 所有衍生物從源頭長出來，絕不從下游反推
- **信噪比 > 總量** — 精準的少量 context 優於含噪訊的大量 context
- **回饋迴路決定自主上限** — 測試、lint、型別檢查是 Agent 的「眼睛」
