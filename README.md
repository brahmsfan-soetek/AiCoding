## 核心觀點

- **AI 做執行，人做判斷** — LLM 在所有層次上的「判斷」能力都有明確邊界，超出邊界會退化為基於訓練資料頻率的猜測
- **傳達偏差** — LLM 每次處理都是有損且隨機的轉換，串接越多輪偏差越大；每次 session 必須從源頭文件載入，不基於上次產出接著做
- **信噪比 > 總量** — 精準的少量 context 優於含噪訊的大量 context
- **回饋迴路決定自主上限** — 測試、lint、型別檢查提供明確的 pass/fail 信號，LLM 能直接對照修正；沒有這些硬規則，LLM 無從判斷自己做錯了
